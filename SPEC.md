# 心灵宫殿 - 个人珍藏网站规范

## 1. 概念与愿景

一个沉浸式的西欧宫殿风格个人珍藏网站，如同漫步于梦幻的紫色宫殿长廊。网站以"心灵宫殿"为主题，将六大人生珍藏模块比作宫殿的不同厅堂，每个模块都有独特的宫殿房间与之对应，营造出典雅、神秘而温馨的珍藏感。

## 2. 设计语言

### 美学方向
西欧宫廷复古风格，结合神秘紫色基调。参考法国凡尔赛宫与维多利亚时代图书馆的典雅氛围，融入魔法学院的神秘感。

### 色彩系统
```
主色(Purple Majesty): #6B4C9A - 宫殿主墙壁
次色(Imperial Violet): #8B6CAB - 装饰元素
强调色(Golden Accent): #D4AF37 - 皇室金边/高亮
深紫(Night Purple): #2D1B4E - 背景深处
浅紫(Mist Purple): #E8E0F0 - 文字/卡片底色
文字主色(Ivory): #FAF8FC - 主要文字
文字次色(Stone): #9A8DA8 - 次要文字
```

### 字体
- 标题: "Cinzel Decorative", "Cinzel", serif - 皇室感
- 正文: "Cormorant Garamond", "Georgia", serif - 优雅可读
- 中文: "Noto Serif SC", "SimSun", serif

### 空间系统
- 基础单位: 8px
- 间距: 8px, 16px, 24px, 32px, 48px, 64px
- 圆角: 4px (小元素), 8px (卡片), 16px (大容器)
- 阴影: 多层次柔和阴影营造深度感

### 动效哲学
- 页面过渡: 淡入效果，opacity 0→1, 600ms ease-out
- 卡片悬停: 轻微上浮 + 金边发光，transform translateY(-4px), 300ms
- 装饰元素: 缓慢浮动动画，营造梦幻感
- 导航切换: 平滑颜色过渡，200ms

### 视觉资产
- 图标: 使用装饰性SVG图案（皇冠、羽毛笔、书本、酒杯等）
- 装饰: 顶部/底部宫殿纹饰边框
- 背景: 渐变紫色背景 + 微妙的纹理

## 3. 布局与结构

### 整体布局
- 顶部: 宫殿风格纹饰标题区
- 中央: 6个模块卡片区（3x2网格布局）
- 底部: 宫殿风格脚注

### 响应式策略
- 大屏(>1024px): 3列网格
- 中屏(768-1024px): 2列网格

## 4. 功能与交互

### 主页功能
- 展示6个模块卡片，每个卡片包含：
  - 模块图标（SVG装饰）
  - 模块名称
  - 简短描述
  - 悬停时显示"进入"提示
- 点击卡片跳转至对应模块页面

### 模块页面通用结构
- 返回主页按钮（左上角）
- 页面标题（模块名 + 装饰）
- 内容区域（各模块独立内容）
- 底部返回导航

### 交互细节
- 卡片悬停: 上浮 + 金色边框发光 + 阴影加深
- 导航链接: 下划线动画
- 页面加载: 淡入动画
- 返回按钮: 悬停时颜色变化

## 5. 组件清单

### 头部标题组件
- 大标题 + 副标题
- 装饰性边框/纹饰
- 状态: 默认显示

### 模块卡片组件
- 尺寸: 280px x 320px
- 包含: 图标、标题、描述、快捷键提示
- 状态: 默认、悬停（发光上浮）、点击

### 导航按钮
- 皇室风格边框
- 状态: 默认、金色边框，悬停时填充色

### 装饰边框
- SVG花纹装饰
- 用于页面顶部和底部

## 6. 技术方案

### 技术栈
- 纯HTML5 + CSS3 + Vanilla JavaScript
- 无框架依赖，单文件部署
- Google Fonts CDN 加载字体
